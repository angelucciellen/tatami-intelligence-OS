# analyzer/

Batch analysis script for vX Rhythm Engine Phase C.

## Files
- `analyze.py` — main analysis script
- `README.md` — setup and usage instructions (see vX_Analyzer_README.md in docs/)

## Quick start
```bash
python3 analyze.py --csv ../data/YOUR_BATCH.csv --zip ../data/YOUR_BATCH.zip --batch 4
```

## Requirements
pip3 install anthropic pillow pandas numpy
