HTML replay tagger — vX Tagger v2.

## File
`vx_replay_tagger_v2.html` — open in browser with TradingView Bar Replay

## Version lock
Tagger v2 is locked for Phase C. Any update requires:
1. Version bump in filename (v3, v4...)
2. Note in batch log
3. Review whether prior events need re-checking

## Storage rule
Export CSV before accumulating more than 50 events.
Do not rely on tagger internal storage as the only copy.
